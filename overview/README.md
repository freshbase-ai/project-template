# Overview

This directory contains the project overview, serving as the entry point
to understand scope, modules, and their interactions.

## Suggested Structure

- `README.md` - Executive project overview
- `glossary.md` - Domain terms and concepts
- `data-flows.md` - Data flows between modules/systems
- `stakeholders.md` - Stakeholders and their responsibilities
