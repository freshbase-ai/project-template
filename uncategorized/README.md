# Documentos Não Categorizados

Esta pasta é o ponto de partida para documentação de projetos que já estão em andamento mas ainda não têm estrutura organizada.

## Como Usar

1. **Comece aqui:** Coloque notas iniciais, rascunhos, transcrições brutas
2. **Refine gradualmente:** Conforme a documentação amadurece, mova para pastas apropriadas:
   - `features/` - Especificações de funcionalidades (BDD)
   - `specs/` - Especificações técnicas
   - `decisions/` - Architecture Decision Records
   - `meetings/` - Notas de reunião organizadas

3. **Use templates:** Quando tiver clareza, use os templates em `.config/templates/`

## Dica

Não se preocupe em ter tudo perfeito desde o início. O importante é:
- ✅ Capturar informações
- ✅ Versionar com Git
- ✅ Refinar incrementalmente

**Este é um workspace vivo** - a documentação evolui conforme o projeto.
